# Projetos-Java

Este repositório contém pequenos projetos e exercícios que estou desenvolvendo para praticar Java e melhorar minha lógica de programação.

A ideia é manter tudo organizado por pacotes, onde cada um representa um conceito específico da linguagem.

🎯 Objetivo

Praticar conceitos fundamentais de Java

Melhorar lógica de programação

Aprender organização de código

Construir um histórico de evolução no GitHub

🛠 Tecnologias

Java

IDE utilizada: IntelliJ IDEA

▶ Como executar

Clone o repositório

git clone https://github.com/seu-usuario/seu-repositorio.git

Abra o projeto em uma IDE Java.

Execute as classes que possuem método main.
